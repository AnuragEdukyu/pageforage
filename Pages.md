# Pages to Build

| Route | Page Name | Purpose |
| :--- | :--- | :--- |
| `/` | Landing/Home | Intro for writers, explaining the value proposition. |
| `/upload` | File Drop Zone | Real parsing for `.md`, `.pdf`, `.csv`, and `.xlsx`. |
| `/processing` | AI Processing | Fake progress (transition to dynamic preview). |
| `/preview` | Dynamic Preview | Displays real parsed content & metadata from the uploaded file. |
| `/admin` | Simple Layout Config | UI to tweak the output style and global settings. |

## Level 1 Goals
- Non-technical person can navigate the full flow.
- Beautiful, obvious UI.
- Real parsing (raw level) for major formats.
- Previews real data without AI hallucinations.
